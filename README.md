# Plataforma de Inscrição de Cursos (GitHub Pages)

Plataforma estática (HTML/CSS/JS) para inscrição em cursos.

## Como publicar no GitHub Pages
1. Crie um repositório e envie estes ficheiros.
2. No GitHub: Settings → Pages
3. Source: Deploy from a branch
4. Branch: main / root
5. Aguarde e aceda ao link do Pages.

## Uso
- `index.html`: formulário de inscrição
- `admin.html`: ver inscrições e exportar CSV (neste dispositivo)

## Nota
Por ser estática, as inscrições ficam no localStorage do navegador.
Se quiser recolha centralizada (Google Sheets / Firebase / Supabase), peça a versão com backend.
